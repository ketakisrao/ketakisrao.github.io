Problem Identification

UI Evaluation

User feedback

I was communicated that the users found it difficult to navigate around and significant features were masked behind buttons and accordions which made them difficult to discover without a prior demo. A detailed survey led to the following points:

  • The map is congested with pins that don't make much sense. There is no way to determine which pin represents corresponding location.
  • Congested UI; Too many operations filled in one place.
  • The map size is small, therefore it is difficult to plot devices.
  • Many features are abstracted and are not easy to carry out without prior knowledge.

Solution

Problem correction

Restructuring Inventory Hierarchy

After understanding the user's requirements, we finalized the following types of inventory:
  1. Location: A point on a google map denoting the physical co-ordinates of the inventory.
  2. Site: An image of a site layout containing multiple devices.
  3. Device: A physical device that logs data which is required to compute our metrics.
  4. Zone: A cut out from a site, which is a special area of interest for data & metric visualization.

Inventory Addition

Adding Locations

Improvements made
  • Added location categories so as to distinguish between inventory types and quick access
  • Map area increased at the time of addition so as to have wider area for plotting
  • Enabled searching with address input box to speed up the location plot process.
  • Pins are made draggable to adjust location easily & precisely.

Adding Sites

Improvements made
  1. Widened site map area to precisely edit site image.
  2. Added image editing tools so that site images can be edited on the go, one need not search for editors to crop and rotate images.

Adding Devices

Improvements made
  1. Enabled zooming(bottom right corner) into images for precise plotting of device pins.
  2. As users need to add multiple devices at a given time, the page doesn't refresh/go back to the display inventory page unless you hit "Done".
  3. Implemented draggable pins to precisely adjust the pins.

Adding Zones

Improvements made
  1. Zoom and Crop options to easily cut out zones.
  2. Zones can be cut out from a minimum of three devices and hence added appropriate error messages, refer below
